在她出现之前，我就早已知道了她的礼物是什么，毕竟我可一直还记得我们之间的那个加密手段。只是，我却不知道站在我面前的那个人就是她。

当我终于从她冰冷的叙述中了解她的身份时，她却说：“如果你成功了的话，你就可以永远不再记得我的存在了，如你所愿。”

把自己曾经最好的一段回忆抹掉，去投身于被虚假的快意充斥的虚幻世界，真的是我想要的吗？而你为什么一定要像这样用自我残害的方式去遏制对于情感的需求呢？

“我解出来了，但比起你的礼物，我更需要你。”
