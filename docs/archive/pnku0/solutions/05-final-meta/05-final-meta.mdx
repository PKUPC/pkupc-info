# Finalmeta

## 情况简介

如果需要完整理解整个剧情解出 Finalmeta，需要阅读零推和四个 Intro，以及 Finalmeta。

在微信公众号上，每解出一道前四个区的 Meta 的时候，随着下个区的解锁口令以外，还有一份 Finalmeta 解锁口令的一部分。

将四个区 Finalmeta 解锁口令拼在一起即可查看 Finalmeta 页面。

将 Finalmeta 解开后可以进入 Ending。

## True Ending

在 Ending 的微信文章中，如果点击了【查看原文】链接的话，可以跳转到一个隐藏的页面，上面有一些关于主题图的问答。

在隐藏页面的最底部，有一道小题目，通过将 Finalmeta 的答案进行一些变换得到解锁 True Ending 的口令。
