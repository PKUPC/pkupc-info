# 剧情回顾

:::info
如果需要完整理解整个剧情解出 Finalmeta，需要阅读零推和四个 Intro，以及 Finalmeta。

本页在此摘录与 Finalmeta 相关的剧情。
:::

## 第零次推送的序言

<span className="text-violet-400">只要在存档点间无数次地重蹈覆辙，散落在满世界的记忆碎片也终能合而一体，赠予我一个由满世界的妄想谱写的新的幻境<span class="text-violet-600 bg-lime-400 font-bold">宇宙</span>。</span>

## 第一次推送

<div className="indent-8 text-violet-400">
<span class="text-violet-600 bg-lime-400 font-bold">“谜题和钥匙。</span>它们之间是一一对应的。正如只有若合符契者才能读懂对方的心思，两者互相依赖，缺一不可。”

……说那句话的时候，黄昏的余晖正好能在她的微笑里完全溶解——好吧，这确实是个只有在花园呆过的人才能想出的神奇比喻。

我们甚至也约定了一种只有我们自己知道的加密手段。她在高考前狡黠地笑着对我说：“<span class="text-violet-600 bg-lime-400 font-bold">为了保证这个加密手段还有用武之地，我们之后还是去同一所城市吧。</span>”
</div>

:::info
上述两部分分别代指了 Finalmeta 的答案 “P&KU”（Puzzle and Key Universe）和加密的方法（城市地铁线路图）。
:::

## A 区序言

<div className="text-violet-400">
再次睁开眼睛的时候，你又身处于一样的环境——从始发站出发的8号线地铁，眼前出现了“Chapter A”。你突然发现这似乎是一个有着存档功能的游戏系统……等到一章未通关的时候会持续返回存档点，直到…直到生命从这无休止循环中解脱出来？

还在想着，地铁站台上的灯箱上忽然浮现出一行小字：“<span class="text-violet-600 bg-lime-400 font-bold">目标地点在左口袋的便条上。</span>”检查了口袋，果然有张便条，虽然内容很令人茫然，但似乎在暗示着你应当前往的正确地点。

<span class="text-violet-600 bg-lime-400 font-bold">或许你应该抓住那些闪现的记忆碎片……</span>
</div>

:::info
主角在同一条地铁线路上轮回，每次读档前会闪现一些记忆碎片，而目标地点需要通过记忆碎片的内容（meta答案）和便条得到。
:::

## C 区序言

<div className="text-violet-400">
你本以为这会是西西弗式的无穷无尽的任务，但是口袋中的便条告诉你这个世界的造物者只设计了四章，甚至每一章的内容都是一样的，都是<span class="text-violet-600 bg-lime-400 font-bold">从一个一直运转的地铁路线的某个站点出发，沿着另一条线到目标站点。不同的只是目标站点和初始路线。</span>
</div>

:::info
目标站点是沿着“另一条线”可以到达的。

充分理解了设定，我们就可以继续看 Finalmeta 了。
:::
