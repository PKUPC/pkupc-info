# C1

## 题面

一节文学课上，讲台上被杂乱地放上了这些碎纸片，黑板上是用记号笔写下的句子：<span className="text-red-600">“Find ALL Numbers” </span>

![](./01-c1.assets/image_1.webp)

## 答案

<AnswerBlock>02334011</AnswerBlock>

## 解析

“Find ALL Numbers”有两层含义。第一层非常显然，就是纸片上所有的“一”。

接下来如果把任意文案丢进搜索引擎，就会发现这是中v的歌词。于是转而用音乐软件获取下述信息：

- <span className="text-red-600">**一**</span>束光线在我眼前消散（《〇》，词作ilem）

- 被赐予<span className="text-red-600">**一**</span>场爱的审判（《二十三》，词作JUSF周存）

- 莫叹息，我再舞<span className="text-red-600">**一**</span>曲你意乱情迷（《三月雨》，词作圣月樱泪）

- 就像你在背后做过的<span className="text-red-600">**一**</span>样（《四角游戏》，词作雨狸）

- 想用我最后的自由策划<span className="text-red-600">**一**</span>次或许悲凉的逃脱（《零重祈愿》，词作雨狸）

- 在这墨雨色的空中混入了<span className="text-red-600">**一**</span>点点的泪（《一秒之序》，词作PoKeR）

- 不想用复仇这词来污蔑，我只是在收复本属于我的那<span className="text-red-600">**一**</span>切（《单向宣战》，词作JUSF周存）

于是，第二层意思就是指标题里的数字，按照“一”的位置顺序，连接所有标题的数字得到答案 **【02334011】**
