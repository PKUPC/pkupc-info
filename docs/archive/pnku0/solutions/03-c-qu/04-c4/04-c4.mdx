# C4

## 题面

老师在展示自己的摄影作品时，相片集中不小心掉出一张奇怪的明信片，背景是四个女生学生时代的合照……花朵艳丽终散落，谁人世间能长久——她们现在又如何呢？

咦？明信片的底下还有些问号。<span className="text-red-600">这八个问号是什么？</span>

![](./04-c4.assets/image_1.webp)

## 答案

<AnswerBlock>01233430</AnswerBlock>

## 解析

这道题的突破口有两个，一个当然就是背景图（《花开伊吕波》，知道了其实也没什么用，充其量是crosscheck——除非预先知道伊吕波歌），还有一个就是那句奇怪的话。

塞进搜索引擎。

![](./04-c4.assets/image_2.webp)

发现伊吕波歌正好是四行，而且是日语的**全字母**句。这就好办了。

![](./04-c4.assets/image_3.webp)

填入。下面要解决的是右侧是什么——显然是日语的**字母表**五十音图。这时候就意识到下面的哪些符号是指左侧的对应位置的假名和右侧对应位置的假名是相同的。

据此得到答案 **【01233430】**
