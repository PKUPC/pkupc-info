# D 区

## 情况简介

D 区在设计上难度较高，尤以 D4 为最，但在选手反馈时发现这个区域的难度分化严重。metaD 的设置提供了一定的容错率。

D 区一共有六道小题和一道 meta，触发口令分别是：

- D1断页草稿
- D2文案碎片
- D3思绪隔阂
- D4信封旧忆
- D5字符乱码
- D6昔日残章
- metaD曲径通幽
