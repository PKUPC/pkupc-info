# 关于 P&KU

P&KU，如名字所见，与一所叫作 PKU 的大学很显然脱不开干系。这是一个由北京大学的学生 Winfrid 策划的解谜活动。
在活动中，玩家可以在解谜的期间不断推进着剧情，还原属于故事主角们的记忆篇章。

**剧情**：P&KU 的剧情大约由 50% 的少女贴贴和 50% 的悬疑构成。
正如[主题曲《春风衔笺》](pnku2/theme-song)中所讲述的，故事的主角是芈雨和觅月，分别谐音着“谜语”和“密钥”（这个字可以读 yuè！）。  
P&KU，全称是 Puzzle and Key Universe，其也正是属于芈雨和觅月、谜语和密钥的宇宙。

**作品**：目前，P&KU 系列已经举办了四届，分别是 [P&KU0](pnku0)、[P&KU1](pnku1)、[P&KU2](pnku2) 和 [P&KU3（上）](pnku3-part1)。其中后三个活动的存档站你可以在页面底部的“往期活动”中找到。  
特别地，在刚刚过去的 P&KU3（上）中，我们在正式比赛结束后还开放了一个[可以游玩的环境](https://pnku3-playground.pkupuzzle.art)，你仍然可以拉上好友一起注册账号并组建队伍，
体验原汁原味的 P&KU3（上）。

**谜题**：P&KU 系列的谜题老少咸宜，它不会涉及专业的密码等学科知识，反而会有着如 LoL、动画、轻小说、Vocaloid、现代诗、推理小说、PKU 课程/树洞、地铁、曲艺、漫才、二游、AIGC 等多种题材。比如，在之前的比赛里就有着以[《弈》](pnku1/solutions/jing-li-guan-hua-zhe-de-jian-ying/jiao-shi-yi)、[《繁华唱遍》](pnku1/solutions/zhen-zhuo-ci-ju-zhe-de-yin-yu/fan-hua-chang-bian)、[《横竖撇点折》](pnku2/solutions/mi-yue-qu/wu-er-diao-gou-hua-bian-chui)为背景的谜题！

总而言之，P&KU 是 Winfrid 的主要解谜作品系列。但比起单纯的解谜，它包含了更多的剧情要素，可以认为是 Winfrid 的 OC（原创角色）们的活动舞台。
